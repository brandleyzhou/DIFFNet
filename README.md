# DIFFNet


This repo is for **[Self-Supervised Monocular DepthEstimation with Internal Feature Fusion(arXiv)](https://arxiv.org/pdf/2110.09482.pdf), BMVC2021**

 A new bockbone for self-supervised depth estimaiton.
